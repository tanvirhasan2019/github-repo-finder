# GitHub Repository Finder

A simple React app to search GitHub repositories using the GitHub Search API.

## Screenshot

![App Screenshot](./src/screenshot/git-repo-finder-ui.png)

## Technology

- **TypeScript**
- **React.js**
- **Material-UI (MUI)**
- **GitHub Search API**

## Installation

```bash
npm install
```

## Run the App

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build

```bash
npm run build
```
